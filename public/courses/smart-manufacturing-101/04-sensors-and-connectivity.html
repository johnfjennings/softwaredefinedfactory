Vibration Monitoring

The #1 starting point for predictive maintenance

  • What it detects: Bearing wear, shaft misalignment, imbalance, looseness
  • Sensor type: MEMS accelerometers (low cost) or piezoelectric (high precision)
  • Sampling rate: 1-25 kHz typical for rotating equipment
  • ROI: One prevented bearing failure can save $10,000-$50,000 in downtime

Temperature Monitoring

Critical for process control and equipment health

  • Thermocouples — Wide range (-200 to 2300°C), rugged, inexpensive
  • RTDs — Higher accuracy, narrower range, more expensive
  • Infrared (IR) — Non-contact measurement, great for moving parts
  • Thermal cameras — Full thermal imaging for electrical panels, machinery

Current and Power Monitoring

The easiest and cheapest way to start monitoring equipment

  • CT clamps — Non-invasive, clip around power cables
  • No machine modification needed — Install in minutes
  • Detects: overloading, idle time, cycle patterns, anomalies
  • A motor drawing 15% more current than normal is a warning sign

Best first sensor for brownfield factories

Wireless Protocols

  • Wi-Fi — High bandwidth, high power consumption, existing infrastructure
  • Bluetooth/BLE — Short range (30m), low power, good for asset tracking
  • LoRaWAN — Long range (5-15km), very low power, low data rate. Great for environmental sensors
  • 5G/Private LTE — High bandwidth + low latency. The future of industrial wireless
  • Zigbee/Thread — Mesh networking, good for dense sensor deployments

Edge Computing

Not all data needs to go to the cloud

  • Edge gateway — Processes data locally, sends summaries to cloud
  • Reduces bandwidth costs by 90%+
  • Enables real-time alerts without internet dependency
  • Popular options: Raspberry Pi, Siemens IOT2050, AWS Greengrass

Rule of thumb: Process at the edge, analyze in the cloud

Choosing Your First Sensors

A practical decision framework:

  1. What's your biggest cost? Downtime → vibration. Energy → CT clamps. Quality → vision.
  2. New or existing equipment? New → wired + OPC UA. Existing → wireless retrofits.
  3. Budget? Start with <$1,000 in sensors. Prove value before scaling.
  4. IT support? Cloud platform if yes. Standalone edge device if limited IT.
